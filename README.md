# AI implementation skill for https://shakespeare.diy

Skill for use on https://shakespeare.diy that enables easy implementation of AI

## How to add the plugin

Open the three-dot-menu (or rather three-line-menu) on the upper left, then click on **Settings**. Then click on **AI**. Scroll down and click on **Advanced**. 

Then scroll down to **Plugins** and click on **+ Add Plugin**. 

Enter the following URL in the text field: `https://github.com/stl1988/shakespearediy-ai-implementation-skill`. Then click the purple **Add Plugin**.
